Getting Started

1
Sign up for a kwtSMS account at kwtsms.com
2
Get your API credentials from your kwtSMS account settings (not your mobile number)
3
Go to the Settings tab above and enter your API username and password
4
Click Test Connection to verify your credentials work
5
Select your Sender ID from the dropdown (must be pre-approved by kwtSMS)
6
Enable the gateway and create a HubSpot workflow with the "Send SMS via kwtSMS" action

Important Notes

  • KWT-SMS is a shared test sender. Register your own Sender ID for production use.
  • Arabic messages use 70 characters per page (vs 160 for English).
  • Test mode sends hit the real API but messages are not delivered to handsets.
  • Delete test messages from your kwtSMS queue to recover held credits.
  • There is a 15-second minimum between messages to the same number.
