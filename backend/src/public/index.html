Setup

Up and running in 3 minutes

1

Install the App

Find kwtSMS in the HubSpot Marketplace and click Install. No payment required.

2

Connect kwtSMS

Enter your kwtSMS API credentials. The app verifies your account and loads sender IDs and coverage.

3

Create a Workflow

Add "Send SMS via kwtSMS" as an action in any contact-based workflow. Pick the phone property and write your message.

4

Start Sending

Done. Every trigger fires the workflow and sends SMS to your contact automatically.
